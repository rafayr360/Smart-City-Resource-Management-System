# 🚀 Smart City Resource Management System (SCRMS)

A Smart City Resource Management System (SCRMS) built in **C++** that demonstrates the practical implementation of **Data Structures and Algorithms (DSA)** to solve real-world urban management challenges.

The project focuses on improving emergency response, resource allocation, citizen management and service scheduling through efficient algorithms and optimized data structures.

---

## 📌 Overview

SCRMS is designed to simulate how a smart city can efficiently manage its resources and respond to emergencies. The system integrates multiple data structures and algorithms to provide an organized and scalable solution for city administration.

---

## ✨ Features

- 🚑 Priority-based emergency response management
- 👥 Citizen information management
- 📦 Efficient resource allocation
- 📅 Smart service scheduling
- 🛣️ Shortest route calculation
- 🌐 City map traversal and connectivity analysis
- ⚡ Fast search, insertion and deletion operations

# 🚀 Getting Started

## 📋 Prerequisites

Before running the project, make sure you have:

- C++ Compiler (GCC, G++, or MSVC)
- Code::Blocks or Visual Studio (recommended)
- Git (optional)

---

## 📥 Clone the Repository

```bash
git clone https://github.com/rafayr360/Smart-City-Resource-Management-System.git

cd Smart-City-Resource-Management-System
```

---

## ▶️ Run Using Code::Blocks

1. Open **Code::Blocks**
2. Open the project or `main.cpp`
3. Click **Build**
4. Click **Run**

Or use the shortcuts:

```
Build  → Ctrl + F9
Run    → Ctrl + F10
```
